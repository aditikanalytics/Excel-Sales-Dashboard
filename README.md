# 📊 Excel Sales Analysis Dashboard

## Project Overview

An interactive Sales Analysis Dashboard built in Microsoft Excel to monitor sales performance, profitability, and customer behavior from 2014 to 2017. This project transforms raw sales data into meaningful business insights through dynamic visualizations, helping stakeholders identify trends, top-performing categories, and profitable customers.

## ✨ Key Features & Visualizations

- **Interactive Filters**
  - Dynamic slicers for **Category** (Furniture, Office Supplies, Technology)
  - Year-wise filtering from **2014–2017**

- **Sales Distribution Analysis**
  - Sales by Category
  - Sales by State

- **Trend Analysis**
  - Profit Gained Over Time
  - Monthly Sales Performance Trends

- **Customer Insights**
  - Customer Count Growth Analysis
  - Top 5 Profit-Generating Customers (e.g., Tamara Chand, Raymond Buch)

## 📂 Data Structure

The workbook is organized into dedicated sheets for data preparation and analysis:

- `sales data`
- `Profit Gained`
- `MonthlySales`
- `Top5Customers`
- `SalesByState`
- `CustomerCount`

All analytical components are integrated into a centralized **Dashboard** sheet for an interactive reporting experience.

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## 📈 Business Value

This dashboard enables data-driven decision-making by providing a clear view of sales trends, profitability patterns, regional performance, and customer contributions over time.
